							DLC Bundle Rare Weapons Set A contains the following items:

A set of hard to get weapons in one bundle. Includes Moro's Masatoshi, Kotobuki's Tearnado, Akagi's Garuda, Kogure's Soul Culler and J.J.'s Joanna. With this set, these weapons are purchasable from the specter in the cemetery.
