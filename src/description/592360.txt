
							About This Game
							Story"Violet's Dream" (Chinese name "紫嫣的奇幻梦境") is a simulation to run cool and fly-based VR experience game, accompanied by real-time strong QTE system to enhance the player in the game scene empathy.HTC VIVE OperatingThe game needs to use the HTC VIVE device helmet with the handle.Players can shake the handle to simulate the game running.Players can respond to game events by pressing the trigger key.Players can lift the handle by simulating the slide down.
