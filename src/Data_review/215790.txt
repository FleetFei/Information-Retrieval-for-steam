That does the trick for me, but others have complained that it didn't do any good when they tried it, so fair warning. I'm using Windows 7.  There are several resolution settings, although there are a few that I've never run across before.<br><br>The second issue has to do with registration of the game. The form asks for alot of personal information, but once filled out it works for me, but a few have complained that once filled in the program refuses to run until all blocks are 'filled in'. I didn't have this issue.  I've noticed that the game has a tendency to crash if I inadvertently hit the Space Bar when the action gets frantic. Normally the Space Bar is used for the Plunger. If I can avoid that mistake all is well, although it has happened a few times, usually when I have an enormous score built up.<br><br>There are six tables, each with a different theme, and various difficulty settings. Novice. Normal. Hard. Expert. I wish the tables were brighter, but that may just be my personal prefrence. The tables are good, but aren't up to the standards set by Zen studios with it's Pinball FX2 game. Plus Zen offers 57 tables with more features and are about to release more. Pinball FX2 sells it's tables as DLCs with one free table. Dream Pinball 3D has no DLC stuff to buy. Many complained (not me) when a bug in the game forced Zen to reset the Leaderboards (not the Local scores though), but Dream Pinball 3D doesn't even have a Leaderboard. Just local scores. You can upload your scores with Dream Pinball 3D, but I have no idea where they go when you do so. This function fails to work alot of the time anyway.<br><br>This game is alot of fun though, especially if you like old-school pinball. The game itself is about 8 years old so it's starting to show it's age. The ball physics are good and there are actual goals to achieve, unlike Fantastic Pinball Thrills which has neither. The controls take getting use to. The Left Flippers are controlled by the Left Shift key, and the Right Flippers are controlled by the Enter key. Gameplay is solid.<br><br>The High Score screen has scores already entered as default scores to aim for. If you don't at least reach the lowest high score then the game will momentarily display your score beneath the high scores. As you achieve higher and higher high scores the target scores go up. So it becomes more and more difficult to reach the minimum, assuming of course that you never improve.  Learning the goals of each table and how to achieve them is key.  Some tables, like Amber Moon for example, have more than one way to increase the multiplier.<br><br>I like it. Currently 38 friends of mine own this game and 4 want it. I believe I have logged more play time than any of them. This may have to do with the difficulty some have had getting this game to work. Use Compatibility Mode. If you still have issues then try different resolution settings. The game was developed by A.S.K. Homework Game Development, who also developed Zaccaria Pinball for Android devices.
