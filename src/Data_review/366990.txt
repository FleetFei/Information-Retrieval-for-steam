Information<\/div><b>Title: Puzzle Station 15th Anniversary Retro Release<br>Genre: Casual, Indie<br>Developer: Ninai Games , YeaBoing <br>Publisher: YeaBoing\t<br>Release Date: 14 Jul, 2000<\/b><br><br><div class=\"bb_h1\">Basic Overview<\/div>For starters I am not one for puzzle games. But this puzzle game I really enjoyed, it is surprisingly fun to play, although it lacks a few basic things to keep you interested enough to play it for hours on end. It has some okay music, and generic sounds. But for a puzzle game you only really focus on the music and game play.<br><br><div class=\"bb_h1\">Game Play<\/div>When I first played this game it reminded me much of the old game Dr. Robotnik's Mean Bean Machine, with the similar style of game play. There are two types of mode in this game. Arcade mode, and puzzle mode. This does add a little bit of replay value to the game, but still only really kept me interested for about 2 hours before it got a bit boring and started to become a little annoying. The controls you will be using are Ctrl and arrow keys. I'm really not sure why they chose ctrl to be their main button to select things in the main menu and to use in the game. I did have a few issues with the ctrl button, for one it's right next to the windows key for me, and pressing that minimises the game. If you then try to go back on the game it crashes. Really annoying, and couldn't find anyway to change the controls.There are two types of game modes on this game. <b>Arcade mode<\/b> is basically connecting 4 of the same coloured blocks, that will get rid of them. Each level you have to get rid of a certain amount of blocks to pass the level in the time limit that is given to you. The first few levels that I played I didn't even realise that there was a time limit that I was up against. Obviously I found this out after a few levels, and this time constrain gets pretty tight on the later levels that you get into the game. There are a few elements to this game, that are again kind of similar to the old Dr. Robotnik's Mean Bean Machine game. There are brick type blocks that if your 4 blocks (or more) touch the brick block it will take it out with them, or they will fall if they are not taken out and if they have space to fall. If however there are solid blocks, they cannot be gotten rid of and doesn't have gravity to it either. These two simple elements of the game make this game get harder as it goes along. Along with the time constrain it makes it a nice challenging game that has a nice difficulty curve to set you into what the game has to offer, and doesn't drastically increase the difficulty which is really good for this type of game. The other mode that there is is called <b>Puzzle mode<\/b>, it is what it says it is. Basically you have a limited amount of turns to drop the blocks, and you must clear all of the blocks to pass the level. I felt like some of the later levels where way too easy then some of the levels before that one, this makes me feel clever but kind of defeats the object of a puzzle game. Both of these modes have a nice little kind of collectible, you can unlock different colours of like the hand held console thing that you play these modes on. Nice little feature.<br><b>Score (Out Of 10): 8 - Good variety of modes, Arcade and puzzle, although in puzzle mode some of the puzzles seem too easy later on and too difficult early on. Also I would defiantly change the main button to select things from ctrl to something else<\/b><br><br><div class=\"bb_h1\">Graphics\/art style<\/div>The Art style of this game is cute, and fits in with the whole puzzle game on like a old school hand held device. There isn't really much else to talk about for the Art Style of this game.<br><b>Score (Out Of 10): 10 - Very fitting graphics<\/b><br><br><div class=\"bb_h1\">Music &amp; Sound<\/div>The music for this game is not bad, although there is only one track that repeats itself over and over again. It really needs to be varied when it comes to the music because after about a hour of listening to it I decided to just mute my sound so I wouldn't have to listen to it anymore. As for the sounds they are pretty generic for this type of game, nothing else really to comment about or say.<br><b>Score (Out Of 10): 6 - Really annoying music if you're playing in a long session, sounds are pretty generic.<\/b><br><br><div class=\"bb_h1\">Conclusion<\/div>I defiantly recommend this game to puzzle fans, and people who don't even like puzzle games. I myself I don't really like puzzle games but enjoyed my time with this game.
