This may of course have changed by the time you read this, but if not, I would advice to simply temporarily uninstall JRE8 for the duration of playing the game. Some 20 hours, that is.<br><br>Note that if you insist on not even being offered the upgrade to JRE8 by the JRE7 you will find the relevant option in the Java 32-bit control panel to not stick unless you run it as administrator. If you run Steam and the game under an unprivileged user account, temporarily <b>give<\/b> this user administrative rights, only to unselect automatic updates as per the above. And blame Oracle.
