I'd be more interested in content more like the main game than little, &quot;inspired by,&quot; mini-game sets.<br><br>Which brings me to my core point: I like what the main game has going for it. What's there is good, if needing a little polish. As is, I'd give it a solid 7-7.5\/10, but with some expansion\/extended content and added polish I would easily raise that score.
