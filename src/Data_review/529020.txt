Obviously A lot of things have been taken into consideration when simulating the running in this game which makes it really convincing.<br>The map handling takes time to get used to however. You can rebind the keys if you want but there are a lot of commands to handle the map. I find it easiest just to use the &quot;focus on current leg&quot; and the &quot;scroll&quot; to manage my map handling. It takes practice however. But all in all, the game really accomplishes a good orienteering simulator, which makes it a great tool for practising in that sense.<br><br>Terrain: Very beautiful. There are a lot of options when it comes to weather and time of day which is really nice when you want to add a nice atmosphere.<br><br>When it comes to map generation: There is currently no map converter implemented and as I understand, there is no promise there will be one available to the ordinary user in the future either. The current maps coincides well with the generated terrain with the exception of some smaller height\/brown details that I found to be missing while playing around. class=\"postedDate\">Posted: December 10, 2016\t\t\t\t\t\t\t\t\t\t<\/div>\r\n\r\n\t\t\t\t\t\r\n\r\n\r\n\r\n\r\n\r\n\r\n\t\t\t\t<div class=\"content\">\r\n\t\t\t\t\tTL;DR&gt; definitively recommended if you are an orienteerer, but way overpriced otherwise.<br><br>Virtual-O is a orientering simulator, and that is all it is. If you know what orientering is, and have done it before, you will get a simulator that is much better than any other orienteering simulator on the market (as those are a decade old). If you do not know about orientering, this game will not work for you, since it requires you to have a lot of detail knowledge about map symbols, control symbols, how to use the compass, and how to do a course. It is a great game to improve map-reading skills and route choice strategies, especially since you get a chance to see how the good players (and there are already some playing this game right now) do it during the replay, which you might know from the TV coverage of the world championship but might never have seen for your own races.<br>There is little help in the game right now for beginners, safe for some very high-level loading screen tips; you will be thrown into a course and the tutorial only covers how player input works. (If you quit the course, you can enable an option that puts the compass to  where you are - which is about the most important thing to get right in the beginning - but that is not very obvious.) This can result in a rather frustrating experience, especially if you take on difficult courses. The game is clearly targeted at good orienteerers - if you are completely new to orienteering, you will have to read a book on the sport in order to be able to play it at all.<br><br>What this game does really well is to simulate how movement in the terrain is impacted by route choice. If you take a shorter path through heavy bushes, you will be slowed down a lot, and also loose your bearing easily. This is much more like actual orienteering (well, to the extent a computer can replicate it) than I've experienced in other simulators. It makes it much more important to do smart route choices and relate the map to your surroundings, and you will not get far by just aligning the compass and running forward all the time.<br><br>Right now, the game offers two maps, which are of pretty high quality. How many more can be added remains to be seen, and there is no community-based map creation yet. In that regard, the game is really early in its development, and it is not entirely clear that this can be solved in the future. The problem is clearly the quality the game already has: a fully automated map import will not produce quality on par with the existing maps. Further manual work is required, for which the developer would have to add a rather complex level editing into the game, or provide it alongside - which might be hard to accomplish. Since courses (on the existing maps) can be added by the community, you can play this game for a while before it gets exhausted, but after a few hours the maps will become familiar. The price-tag of 40\u20ac does not reflect that, and clearly conveys that the game is a special interest game.<br><br>If you know orienteering, and have done some courses in real life, and want to improve your map and route planning skills, get this game. It comes much closer to the real-life experience than the other orienteering simulators, looks much better, and is a great opportunity to compare your route choices to other orienteerers.<br>If you are new to orienteering, this game might be a way to get started if you want to invest the offline learning of map symbols and how to use a compass. You'll have to accept that usability can be a bit tricky then, but there are options on a course re-run that will help you.<br>If you do not know orienteering (and do not care), you should not buy this game now, since it will just give you a frustrating experience and not do anything to get you hooked. As a mere walking simulator, the game won't be fun at all.<br><br>Or looking at the price tag: if you are wondering if upgrading to a headlamp with 6500 lumen will improve your orienteering performance, get this game, it will give you good value for the price. If you want to try out orienteering without getting lost in a real forest, wait a bit till the game gets on sale or has its onboarding improved.
