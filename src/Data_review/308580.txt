2014\t\t\t\t\t\t\t\t\t\t<\/div>\r\n\r\n\t\t\t\t\t\r\n\r\n\r\n\r\n\r\n\r\n\r\n\t\t\t\t<div class=\"content\">\r\n\t\t\t\t\tOrborun is a game that requires skill. It\u2019s a game that can easily be picked up and played for a couple of minutes but it can also be played for hours on end. The kind of game anyone can enjoy. It\u2019s a game about a running (and rolling; or vice versa) robot and your goal is to set the best time and have the highest score.<br><br>This isn\u2019t one of those puzzle games that desperately feels like it has to tell you a story just in order to make sense of it all. The game has several modes - Haste, Puzzle and Hazard \u2013 and each mode seems to have 15 levels. A new category will unlock as soon as you finish one of these. So finishing Haste will unlock Haste 2, which has more obstacles and is generally more difficult. As you probably would\u2019ve guessed already each mode is focused on something specific. So Haste focuses on speed, Puzzle focuses on getting as many points as possible while sometimes figuring out where to go and by flipping switches and Hazard is focused on avoiding obstacles and falling down holes. Of course, your main goal is to still set the best time and score.<br>Overall the levels are enjoyable and require good skills and timing. Finishing a level will also earn you coins that can be used in the in-game store to buy cosmetic items for your robot.<br><br>The game gradually introduces some new objects every few levels like ramps, walls and portals.<br>It makes the game more interesting and difficult at the same time. The camera does get in the way of the action sometimes and I think it would\u2019ve helped if it was placed just a tad higher.<br>Collision detection is also a bit flaky; some of the levels require that you bounce off of walls in order to make a left or right, but bouncing straight into the wall will reset you to the last checkpoint. Confusing and frustrating. So you have to move slightly to the left\/right before crashing into it. I understand that the robot isn\u2019t built for high speed crashes but if you implement a mechanic like this you have to ignore these kinds of things if you ask me.<br>The game also doesn\u2019t always register when you pick up (collide with) an item. I\u2019ve had several times that I was 100% sure I picked something up but the game didn\u2019t process it. For me it\u2019s not that big of a deal but if you want to set the high score then it most certainly is.<br><br>Graphics wise the game is simple. Simple but pretty. The robot is well animated and textured, and the way it morphs into a ball (or the other way around) will look good and cool to anyone who likes Transformers.<br>The level design on the other hand is a bit uninspired. The level is set in a gigantic cube with lots of platforms and each of the game\u2019s categories has one primary color. Haste uses a lot of blue colors, Puzzle has an orange colored look and Hazard uses a good amount of green. It\u2019s certainly distinctive.<br>But then again, the colors aren\u2019t the problem; it\u2019s the fact that the levels feel a bit empty. There\u2019s nothing happening in the background. You can argue with me here and tell me that this is for serious gamers that don\u2019t want to be distracted, but then I\u2019d say that any serious gamer could probably look past those things. I don\u2019t want to see a level full with animated stuff, but a few things wouldn\u2019t have hurt.<br>The soundtrack is electronic and it usually has a good, solid beat. You\u2019ll probably have heard all of it after just a few levels, but I guess most of you won\u2019t even notice because you\u2019ll be too focused on finishing a level. The sounds effects are kept to a minimum and can usually only be heard by collecting items, or by the occasional crash into an object (which is rare).<br><br>Orborun isn\u2019t a very long game. A level usually takes less than a minute to complete. If you can complete it in one turn, that is. A lot of the levels will definitely take a few tries in order to complete them and if you\u2019re a true completionist than it will take you many tries more. Since there are 6 categories consisting on 15 levels each the game should take you at least 90 minutes to complete. And that\u2019s a best case scenario. I think you should add at least a couple of hours more if you want to be truly accurate. And afterwards there are still the Bonus levels to be completed. So all in all there\u2019s a good amount of value to be found here.<br><br>Orborun doesn\u2019t really reinvent the wheel  but what\u2019s there is (mostly) solid, and fun. Sure, there\u2019s always room for improvement but anyone looking for a game to set a new record (time and\/or points) will have a blast with this one.<br><br>[Rating: 75\/100]\t\t\t\t\t<div
