A Big Jack. Very annoying for anyone who doesn't want to use the HUD. You can see this in a screenshot above.<br>- It will not work when connected to &quot;dead&quot; engines. I believe if you put another locomotive between them it does work however.<br><br>Despite the fact that I see the dynamic brake as a massive fall back I'm going to recommend it - but get it on sale. First, it may be that it is realistic, tho two SD40's provide much superior braking force and should be approximately equivalent. Second, almost everything else about it is very good.
