was very excited to learn that CAVE would be releasing another game on Steam.  While I would have preferred Mushihimesama Futari, I wasn't about to turn down the possibility of playing an entry in their longest running shooting game series.  DoDonPachi Resurrection (also known as DoDonPachi DaiFukkatsu) was released in Japanese arcades in May 2008.  Since then it has been released on Xbox 360 and mobile devices.  Perhaps it shouldn't be a surprise that this is the game CAVE chose after having already made it so widely available.<br><br>Presumably in an effort to entice individuals that already purchased the game on other platforms, the Steam edition of DoDonPachi Resurrection is the most complete version available.  Unlike Mushihimesama, there is no gameplay related DLC.  There are an almost overwhelming number of modes including: 1.5, 1.51, Arrange A, Arrange B, Black Label, and Black Label Arrange.  Although there are basic similarities between the modes, each one is essentially a different game.  There is so much content that it's a little overwhelming if you're not already familiar with everything.  The game doesn't offer much of an explanation for anything. I searched the internet to learn about the differences between modes.<br><br>I won't go into a detailed description of everything because it would require pages of text.  The feature that really defines DoDonPachi Resurrection is the hyper ability.  Ships can enter hyper mode for a brief period of time after filling the hyper gauge.  While hyper is active your bullets destroy enemy fire.  This can be used not only for surviving tricky situations, but is also an important part of the scoring system.  If hyper is used excessively it becomes less effective and floating \u201cdanger lasers\u201d will appear and attempt to shoot you down.  Other notable features of the game are secret boss routes and special loops.  Depending on the game mode, there are hidden loops that can be accessed by fulfilling certain requirements. Although the game doesn't explicitly tell you about it, there are a wide range of difficulty  options available.  A simple 1-ALL (a clear without continuing or looping) with strong style should be achievable for newer players while a 2-ALL of the second secret loop with power style will challenge shooting game veterans.<br><br>Whatever your goal is, be it survival or score, it's probably going to take practice.  You'll likely find yourself repeatedly playing the same stages to seek perfection.  Fortunately, all of the OSTs are outstanding.  Black Label in particular has some of the best tracks I've ever heard in a shmup.  The graphics are beautiful as well.  Not only do the ships look cool, but the hit box is made clear by glowing at all times.  As much as I enjoy the graphics I do have a few criticisms.  The game crashed the first time I switched to full screen, though it did work correctly once starting it again.  I also experience quite a bit of screen tearing.  There isn't an in game v-sync option either.  I've read it's possible to toggle in one of the .ini files but haven't bothered to check it out because I'd rather not have the v-sync input lag.  Lastly, the default language on release day was Japanese.  It was easily fixed by going into the options, but it made me wonder if they bothered to test the game at all before releasing it.<br><br>Sadly, the aforementioned graphical\/menu issues aren't the only problems with this port (as of October 22, 2016).  Black Label Arrange has extra slowdown, making that mode easier than it would be under normal circumstances.  The opposite is true in Arrange A which has missing slowdown.  Personally, this didn't bother me because I was primarily interested in the arcade modes and they work fine.  If those modes are a main selling point for you, it might be more of an issue.  I've had a great time with this game so far, and have found it to be one of the most enjoyable shooting games I've played in some time. I'd recommend this game, but with the caveat that two arrange modes do not work properly and may never be fixed.
