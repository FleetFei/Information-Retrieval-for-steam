Demise of Nations: Rome is a turn based strategy game set during the roman times.  You control several units, and have five resources that you have to manage.  As the game progresses, you research things like unit upgrades, buildings, and terrain upgrades.  The basic scenario starts out with Rome at war with a neighbor, and all other contries are neutral.<br><br>     This game needs improvement.  During your turn, it gives no indication of which units need to move.  When you click on a space with an enemy unit as an action, it moves your unit there and only attacks if the enemy unit is still there.  There is no option for resource icons.  The game takes forever to run it's course, and the aforementioned flaws make winning annoying.  The tutorial, though improved since release, leaves much to be desired.  Capture the Flag mode is thouroughly broken.<br><br>     The multiplayer isn't much better.  It takes coins to create a game, which take forever to regenerate.  Half of the time the game isn't created, and a fourth of the time the created game is unjoinable.  Server times are dismal.<br><br>     The game has some pay content, mostly just extra maps and a generator.  I haven't seen the need to spend money on this, but the generator has promise.  <br><br>     But still, this game has potential.  It's still in early access, and Noble Master has done a good job of responding to bugs and likewise problems.  It's a fairly easy 100%, although the dev has said he might add more acheivements.  Plus, besides the extra maps, it's free.  <br><br>      With some improvement, this can be a great game.  I give it 6\/10, so long as the Dev keeps it up.<br><br>     EDIT -- 6\/10\/15<br><br>     After about a month of development since I posted this review, this game has truly improved.<br><br>     The tutorial is much better.  The We Go system is much better, and many of the single player flaws I mentioned are fixed.  Resource Icons are there, which makes it a lot easier to manage your land.  The new acheivements have been added, and some of them are quite difficult.  <br><br>     There have been some other changes too, but I found that playing this time arround was much easier than
