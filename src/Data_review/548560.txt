Just like Hearthstone does things with RNG (such as summoning random minions) which just aren't possible with a paper card game, ManaStorm leverages the unique capabilities of virtual reality to add an extra layer to the game.<br><br>Here are a few examples:<br><br>* You start each game with a bow and a fire arrow (you can get more arrows through cards). This beam spreads damage to anything you point it at for four seconds. There is currently a damage cap, but the idea of allowing the player's skill in handling this spell determine it's effectiveness is exciting to me.<br>* You start each game with a shield, which can be triggered once to reduce damage. Currently it's a one-off and blocks every time you activate it. But adding an element of skill would be possible here as well. Timing your blocks to shield you from attacks and allowing critical blocks would be great gameplay.<br><br>The creator of Manastorm has shown the ability to mix potions on the battlefield as a future feature. This also looks like an exciting feature if done correctly. The key to all of these features is to put the player under pressure to perform them. If you're expected to play cards, assign attacks, shoot arrows and mix potions all in a minute and a half, I can see this being an exciting foundation for good game play.<br><br>The game is now in it's early stages (pre-access that feels like early beta). With a 3D model and voice lines for each card, that's a lot of work and the game is currently quite rudementary in comparison to other CCGs.
