
							About This Game
							ARTDINK "Legend Pack" SeriesThe ARTDINK "Legend Pack" Series is a compilation of ARTDINK's masterpiece works of the 1980s and 1990s. The PC9800 versions of these games can finally be played in Windows. In addition, a Library Mode exclusive to the "Legend Pack" allows you to browse illustrations, graphic and sound data."Lunatic Dawn Legend Pack" is a compilation containing the following titles:"Lunatic Dawn" (original release 1993)"Lunatic Dawn II" (original release 1994)"Lunatic Dawn III" (original release 1998)Lunatic Dawn are non-linear RPG games set in Medieval Eurasia inspired world.This game is only available in Japanese.
