
							About This Content
							This DLC includes 6 new game modes, with their Leaderboard, High Scores ,Game UI and new Effects!
Slash it 2 - Russian Edition will make the "Slash it 2" game display a new UI bar where you can chose between the original game modes (Classic mode) and this downloaded DLC (Russian mode).
In the "Russian mode" the alphabet,insane alphabet, crazy and words mode are using the Russian letters and words instead of English! Test your skill in the Russian language with the new game modes!
Each Russian game mode has it's own unique song where the game reacts to the new beat!
