
							About This Game
							BitMaster is a dynamic and colorful twin-stick shooter which design was inspired by such games as Geometry Wars, Crimson Land and Satellite Reign. Game powered by Unreal Engine 4!You need to destroy the enemies and collect power-ups to improve weapons. Game balance changes periodically, which makes the game unpredictable and difficult to get a high score.BitMaster is suitable for all ages. Average game session is half an hour long.Features:100 waves of enemies8 guns13 perks5 heroes to unlock20 types of enemies5 wicked bossesOnly 1 way to escapeLot of factors that may cause troubles... or make the game easier!
