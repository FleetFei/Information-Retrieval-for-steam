
							About This Content
							The extended edition of Deity Quest adds many new features to the game. The additional features include:All new locations and quests.An epilogue to the story.The Follower Lab which allows you to create entirely new follower types.New follower types which you can only find in the extended edition.This edition includes the base game as well as the extended edition DLC. If you already have the base game, you can buy the Deity Quest Extended Upgrade DLC.Note that some achievements in the base game are only possible with the extended edition.
