
							About This Content
							This DLC gives you all the extra items in the Deluxe Edition. It is intended for those that already have the standard edition but also want the Deluxe content without buying the game again. If you already own The Deluxe Edition then you have all the content in this DLC.DLC includes:A pixel art map with all the boss locations The full soundtrack composed by Karl Flodin A digital booklet that gives some extra information about the story, weapons, and upgrades A few desktop backgroundsImportant: Nothing is changed or added to the actual game. This is all just extra content outside of the game, which can be found in the game's folder.
