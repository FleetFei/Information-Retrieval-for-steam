
							About This Software
							JugiPaint is a painting software with diverse tools for various illustration works. It provides many standard digital painting features like layers, brushes, selections, transformations, etc., as well as some less often like guide layers and panels. The raster based painting and editing capabilities are supplemented with powerful tools for creating vector based line art.     
JugiPaint is the successor of Comicado and is also well suited for creating comics. 

Key features: 
- a versatile brush system which powers a wide range of brushes with a high level of customization 
- vector drawing
- filter layers
- text layers
- guide layers - perspective and simple mannequin
- layered panels for comics creation
- rulers
