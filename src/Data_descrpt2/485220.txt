
							About This Game
							The Orb ChambersThe Orb Chambers is a puzzle physics plat former, the most intense mouse only game, fast, accurate and quick reflexes is the way to play. With over 30 incredibly hard levels and many hours of game-play this game will get you addicted within minutes. The game uses a physics engine which has been refined for the best feeling possible when playing.The game will run at 60fps and will include a hardmode which will add about 150% more gametime to the 2 - 3 hour game depending how good you are. The game is addictive and well worth the price. support from the community is welcome if you find any bugs or issues please post it in a forum.
