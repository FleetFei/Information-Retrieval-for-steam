
							About This Software
							GeoVoxGeoVox is the latest real time world generation software application created by Axis Game Factory, Inc.  GeoVox is a completely new stand-alone Voxel based real-time terrain creation tool. GeoVox instantly creates lush AAA quality environments at the start of the application.  Designed with intuitive game like controls it allows users to test for level playability as they are creating.
